Kebijakan Cookie (Cookie Policy)

Terakhir diperbarui: 28 September 2025

Apa Itu Cookie?

Cookie adalah file teks kecil yang disimpan di komputer atau perangkat seluler Anda saat Anda mengunjungi sebuah situs web. Cookie membantu situs web mengenali perangkat Anda dan mengingat informasi tentang kunjungan Anda, seperti preferensi Anda, untuk membuat kunjungan Anda berikutnya lebih mudah dan situs lebih berguna bagi Anda.

Bagaimana Kami Menggunakan Cookie?

Kami menggunakan cookie untuk berbagai alasan yang dirinci di bawah ini. Sayangnya, dalam kebanyakan kasus, tidak ada opsi standar industri untuk menonaktifkan cookie tanpa sepenuhnya menonaktifkan fungsionalitas dan fitur yang mereka tambahkan ke situs ini.

Kami menggunakan cookie untuk tujuan berikut:

  • Analitik: Kami menggunakan cookie untuk memahami bagaimana pengunjung berinteraksi dengan situs web kami. Ini memungkinkan kami untuk menganalisis dan mengukur lalu lintas, serta meningkatkan konten dan layanan kami. Kami menggunakan layanan pihak ketiga seperti Google Analytics untuk tujuan ini.
  • Periklanan: Kami menggunakan cookie untuk membuat iklan lebih relevan bagi Anda. Mitra periklanan kami, seperti Google AdSense, mungkin menempatkan cookie di perangkat Anda untuk menampilkan iklan yang dipersonalisasi berdasarkan kunjungan Anda sebelumnya ke situs kami atau situs lain di internet.
  • Fungsionalitas: Cookie tertentu diperlukan agar situs web dapat berfungsi dengan baik. Misalnya, kami mungkin menggunakan cookie untuk mengingat preferensi Anda seperti pilihan mode gelap (dark mode).

Mengelola Cookie Anda

Anda memiliki hak untuk memutuskan apakah akan menerima atau menolak cookie. Anda dapat mengatur atau mengubah kontrol browser web Anda untuk menerima atau menolak cookie. Jika Anda memilih untuk menolak cookie, Anda mungkin masih dapat menggunakan situs web kami meskipun akses Anda ke beberapa fungsionalitas dan area mungkin dibatasi.

Karena cara untuk menolak cookie melalui kontrol browser web Anda bervariasi dari satu browser ke browser lainnya, Anda harus mengunjungi menu bantuan browser Anda untuk informasi lebih lanjut.

Informasi Lebih Lanjut

Untuk informasi lebih lanjut tentang bagaimana kami menangani data pribadi Anda, silakan lihat Kebijakan Privasi kami.
